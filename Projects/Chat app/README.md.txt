steps to start the basic chat app:

1) open terminal on the folder

2) run server2.js using node.js => node server2.js

3) open index.html inside ./App/

4) notice inside the terminal that a user has connected

5) you can now type in the terminal /user1 and press enter

6) then type your message and press enter to send it

7) you can also send messages from the client by clicking the arrow next to the input.

8) for every index.html duplication or reload, a new user connects and the corresponding command /userx is necessary to send messages to
